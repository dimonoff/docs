# Logging into Spatium

For security reasons, access to Spatium is limited to authorized users with the appropriate authentication.

To log in to Spatium, follow these steps:

1. Open a web browser. Spatium is a web application. The following browsers have been tested for compatibility and are supported by Dimonoff (the Spatium platform is not yet optimized for use on smartphones and tablets):

* GOOGLE CHROME
* SAFARI
* FIREFOX

2\.      Enter the URL provided by your Dimonoff administrator (e.g., [https://app.spatium-iot.com](https://app.spatium-iot.com/)). The login window will appear.

<figure><img src="../../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>



3. Enter your username and password, then click "Go," or log in using the Single Sign-On (SSO) option for Google or Azure Active Directory (AD) Microsoft if you have already set it up. The second option involves logging in with your Microsoft account credentials. Contact your system administrator for more information.

<figure><img src="../../.gitbook/assets/image (5).png" alt=""><figcaption></figcaption></figure>

4. If you have access to multiple licenses, a list of those you have access to will appear for selection. Choose the one you wish to enter. If you have access to only one license, it will be automatically selected, and this step will not appear.
