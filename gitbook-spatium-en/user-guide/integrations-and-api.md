# Integrations and API

The Dimonoff Mobility solution is based on an open and documented standard API, allowing for free data exchange and simple integration of third-party solutions. Our API is based on the Alliance for Parking Data Standard (APDS recently adopted as the new ISO standard for parking (ISO TS 5206-1).

This API is documented on our Swagger, where you can also try it out before integrating our data according to your needs (e.g., external website). \
It is accessible at [https://apds-api.spatium-iot.com/api](https://apds-api.spatium-iot.com/api). To try and use the Dimonoff Mobility API, you will need an individual token provided by our support team at the start of the project.
