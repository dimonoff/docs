# 2D Plan Visualization

Spatium allows real-time visualization of your parking spots and enables you to dynamically modify their configuration. This feature is available within each zone, where you can view and configure individual spots.

When viewing a parking layout, several features can simplify management:

1. The “Full Screen” mode allows a zone to be displayed on a large screen in an operations center, for example.
2. The “Spots” button provides a list of filters to view the most important information for you. You can filter the layout by layers (spots, equipment, background, etc.), spot types, associated services, as well as equipment types and statuses. Note that new filters cannot be created.
3. The “Equipment” button provides a list of equipment to view relevant information. You can filter the layout by equipment type (barriers, signs, etc.) or status (open sign, closed sign, etc.).
4. The Spatium Toolbox makes it easy to locate or modify features—simply type the keyword for what you're looking to find or modify. For more details, refer to the “Configure Parking Spots” section in the “Inventory” chapter.
5. The “Change Zone” button allows you to switch zones by selecting the desired zone.
6. Edit mode (pencil icon) is available for configuring your spots, equipment, and groups. Access is limited to project administrators. If the button doesn’t appear, contact your sales representative.
7. The interactive legend (list icon) lets you view icons for spots, smart signs, and barriers with explanations.
8. The delay indicator and quick controls allow you to check for any delay in sensor updates and offer quick control options for better visualization.
9. The '?' icon in the bottom right provides keyboard shortcuts and plan controls. For faster configuration of multiple spots, you can use quick controls available in edit mode to duplicate spots, allowing you to quickly create rows or groups of spots. However, each spot will then need individual configuration with the correct fields.
