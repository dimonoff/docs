# Installation

The Spatium platform is hosted on the cloud (SaaS). Users access the software via a website designed for clients, eliminating the need to install the platform on users' computers.

For more information, refer to the Spatium datasheet.

**SaaS Option (Cloud Server Installation)**

Spatium is installed by Dimonoff and hosted on a virtual machine with our partner Digital Ocean. Since their data centers are located worldwide, you can choose a specific geographic location for the installation of your Spatium platform.
