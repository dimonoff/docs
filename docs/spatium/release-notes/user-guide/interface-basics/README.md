# Interface basics

Before using **Spatium**, make sure the following conditions are met:

### 1. Internet Connection

Spatium is a web-based application. An active and stable Internet connection is required to access the interface, load data, and perform actions within the system.

### 2. JavaScript Enabled

JavaScript must be enabled in your web browser.\
JavaScript is required for the proper functioning of the interface (dynamic page display, user interactions, data loading, and action execution).
