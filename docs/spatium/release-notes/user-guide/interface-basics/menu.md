# Menu

The main menu in Spatium is always located on the left side of the platform. By default, it is always collapsed to provide the maximum space in the application. However, you can see more information inside when hovering in as it will expand automatically.

<figure><img src="../../.gitbook/assets/image (12).png" alt=""><figcaption></figcaption></figure>



A configuration menu is always available at the bottom of the main menu and can be seen in full when expanding the full menu sidebar. To do this, simply hover wherever in the main menu.
