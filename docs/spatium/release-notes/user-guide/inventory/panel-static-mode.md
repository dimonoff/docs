# Panel Static Mode

To set a smart panel in Static mode, select it on the layout. You will then be able to view its various information, such as:

* The number of available spots or live information displayed (also represented on the panel's icon on the layout)
* A static mode toggle
* The set of spots assigned to the panel (groups or zones)
* Information about the display linked to the smart panel in Spatium
* The smart panel data source

<figure><img src="../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>



Click the “Static mode” toggle to switch the panel to static/manual mode. The Static Mode toggle box will then expand to give you more options. You can either select a preset message or set a custom static message in the “Custom message” input. Click on “Send” to send it to the smart panel and display it. The linked spots will retain their real statuses on the layout to highlight potentially abnormal behaviors. The panel will keep the static message on unless put back in Automatic Mode by clicking back on the toggle to deactivate Static Mode.
