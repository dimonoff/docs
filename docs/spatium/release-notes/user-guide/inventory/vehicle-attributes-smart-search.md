# Vehicle Attributes Smart Search

When monitored by certain types of cameras, it is now possible to detect the following vehicle attributes when a spot is occupied:

* License Plate
* License Plate Color
* Vehicle Color
* Vehicle Type

It is possible to search for these attributes and know where corresponding vehicles are parked in your parking facility by simply searching for them through the top Spatium Toolbox. Corresponding vehicles will be presented in the result list and hovering over them will highlight the spot on the plan, allowing you to easily identify where they are parked.
