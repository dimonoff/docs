# Contacter le support technique

Concernant les garanties des produits et solutions Dimonoff, veuillez-vous référer aux [termes et conditions générales de ventes](https://www.dimonoff.com/fr/termes-et-conditions/), spécifiées sur le site Internet de Dimonoff.

Pour consulter le support technique de Dimonoff, visiter ZohoDesk. Au besoin, la plateforme vous redirigera vers la personne définie dans le contrat qui assure la première ligne de support.

* Si vous êtes déjà membre de ZohoDesk, veuillez nous contacter via la page web suivante afin de vous connecter à votre compte en utilisant votre adresse courriel ainsi que votre mot de passe : [https://dimonoff.zohodesk.com/portal/fr/signin](https://dimonoff.zohodesk.com/portal/fr/signin)
* Si vous n’êtes pas encore membre de ZohoDesk, veuillez créer un compte ZohoDesk en consultant la page web suivante : [https://dimonoff.zohodesk.com/portal/fr/signup](https://dimonoff.zohodesk.com/portal/fr/signup)
