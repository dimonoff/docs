# Inventaire

Spatium vous permet de configurer entièrement et de personnaliser votre projet de stationnement intelligent afin d’en tirer le plus d’informations et d’en optimiser les performances. \
Ces fonctionnalités sont présentes dans le menu "Inventaire" de l’application. Dans ce menu, il vous est également possible de visualiser en temps réel l’occupation du stationnement.
