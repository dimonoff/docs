# Recherche intelligente d’attributs de véhicule

Lorsqu'ils sont surveillés par certains types de caméras, il est désormais possible de détecter les attributs de véhicule suivants lorsqu'une place est occupée :

* Plaque d'immatriculation
* Couleur de la plaque d'immatriculation
* Couleur du véhicule
* Type de véhicule

Il est possible de rechercher ces attributs et trouver où sont garés les véhicules correspondants dans votre parking en les recherchant simplement via la boîte à outils Spatium.&#x20;

Les places correspondantes seront présentées dans la liste des résultats et le survol de celles-ci dans la liste mettra en évidence leur emplacement sur le plan, afin que vous sachiez facilement où ils sont garés.
