# Mode statique pour panneau d’affichage

Afin de passer un panneau en mode statique, sélectionnez le sur le plan. Vous pourrez alors en visualiser ses différentes informations comme :

* &#x20;Le nombre de places disponibles affichées, ou le message actuellement affiché, également représenté sur l’icône du panneau sur le plan&#x20;
* Un bouton de passage en mode statique
* &#x20;Les ensembles de places assignées au panneau (groupes ou zones)
* Ses paramètres d’affichage et de source de données

<figure><img src="../../.gitbook/assets/image (64).png" alt=""><figcaption></figcaption></figure>



Cliquez sur le bouton "Mode statique" dans le menu flottant pour passer le panneau en mode statique. Celui-ci s’agrandit pour vous laisser le choix de configuration du message statique.

Vous pouvez alors sélectionner un message pré-fait ou bien en personnaliser un, dans la limite de quatre caractères. Cliquer sur "Envoyer" pour envoyer le message au panneau, qu’il l’affiche, et finaliser la configuration du mode statique. Les places gardent leurs états réels sur le plan afin de mettre en évidence des comportements potentiellement anormaux mais ne sont plus afficher sur le panneau, au profit du message statique.

Afin de repasser le panneau en mode automatique pour que celui-ci affiche à nouveau la disponibilité en temps réal, cliquer à nouveau sur le bouton "Mode statique" pour le désactiver et le repasser en mode automatique.
