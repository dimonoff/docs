# Intégration et API

La solution Dimonoff Mobilité est basée sur une API standard ouverte et documentée permettant un échange de données libre ainsi que l’intégration simple de solutions tiers. En effet, notre API est basée sur le standard de l’Alliance for Parking Data Standard (APDS), récemment adopté comme étant le nouveau standard ISO du stationnement (ISO TS 5206-1).

Celle-ci est documentée sur notre _Swagger_, depuis lequel vous pouvez également l’essayer avant intégration de nos données selon vos besoins (par exemple : site web externe), accessible depuis [https://apds-api.spatium-iot.com/api](https://apds-api.spatium-iot.com/api). Afin de pouvoir essayer et utiliser l’API de la solution Dimonoff Mobilité, il vous faudra un _token_ individuel fourni par notre équipe de support au démarrage du projet.
