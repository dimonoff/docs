# Installation

La plateforme Spatium est hébergée sur le cloud (SaaS). Les utilisateurs accèdent au logiciel à l’aide d’un site Internet conçu pour les clients. Ainsi, il n’est pas nécessaire d’installer la plateforme sur l’ordinateur des utilisateurs.

Pour plus d’informations, veuillez-vous référer à la fiche technique (_datasheet_) Spatium.

**Option SaaS (installation sur un serveur cloud)**

Spatium est installé par Dimonoff et hébergé sur une machine virtuelle avec notre partenaire Digital Ocean. Étant donné que leurs centres de données sont situés dans le monde entier, vous pouvez choisir un emplacement géographique spécifique pour l’installation de votre plateforme Spatium.
