# Changer de langue

Le changement de langue est également possible afin de sélectionner entre l’anglais, le français et l’espagnol. D’autres langues peuvent être ajoutées de façon très simple et rapide (contactez votre représentant de vente pour plus d’informations). Par défaut, Spatium se configure automatiquement avec la langue de votre navigateur. Le changement de langue est possible également dans le menu inférieur gauche.
