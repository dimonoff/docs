# Créer un utilisateur

La plateforme Spatium est construite sur la base de Fundamentum. Ce qui lui permet de supporter toute la gestion des utilisateurs fournit par Fundamentum Connect. Veuillez-vous référer à la documentation de Fundamentum Connect pour en savoir plus sur la création d’utilisateurs à la gestion de leurs droits et restrictions.
