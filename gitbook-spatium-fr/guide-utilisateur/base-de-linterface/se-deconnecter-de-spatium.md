# Se déconnecter de Spatium

Une fois que vous avez terminé de travailler sur Spatium, vous pouvez vous déconnecter en plaçant votre souris dans le menu, qui va alors s’ouvrir. Dans la partie basse ce dernier, vous pourrez ensuite cliquer sur le bouton "Déconnexion".
