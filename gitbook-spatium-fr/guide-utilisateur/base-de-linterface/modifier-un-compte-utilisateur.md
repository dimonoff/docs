# Modifier un compte utilisateur

Une fois connecté et naviguant dans la plateforme, vous pouvez à tout moment modifier vos informations de compte utilisateur Fundamentum Connect : nom, adresse courriel (utilisée comme nom d’utilisateur) et mot de passe. Vous pouvez également y configurer votre option de SSO.

Pour cela, il suffit de cliquer sur le bouton "Gérer les utilisateurs" en bas du menu principal.
