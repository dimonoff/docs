# Base de l'interface

Avant d’utiliser **Spatium**, assurez-vous que les conditions suivantes sont remplies :

#### 1. Connexion Internet

**Spatium** est une application accessible en ligne. Une connexion Internet active et stable est requise pour accéder à l’interface, charger les données et effectuer des actions dans le système.

#### 2. Activation de JavaScript

Le navigateur utilisé doit avoir **JavaScript activé**.\
JavaScript est nécessaire au bon fonctionnement de l’interface (affichage dynamique des pages, interactions, chargement des données et exécution des actions).
