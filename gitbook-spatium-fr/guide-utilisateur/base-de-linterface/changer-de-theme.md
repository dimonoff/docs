# Changer de thème

Il est possible dans Spatium pour chaque utilisateur de changer le thème pour passer du mode jour au mode nuit. Par défaut, le thème affiché est celui utilisé par votre ordinateur. Afin de le changer, vous pouvez cliquer sur le bouton "Basculer en mode sombre" en bas à gauche du menu principal.
