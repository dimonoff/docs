# Changer de licence

Après la connexion à Spatium, il est possible de sélectionner la licence à laquelle vous souhaiter accéder. Une fois celle-ci sélectionnée, il vous est possible d’accéder à une autre licence à laquelle vous avez accès dans le menu en bas à gauche de Spatium. Pour cela, passez la souris sur le nom de la licence dans laquelle vous vous trouvez avant de sélectionner celle dans laquelle vous voulez vous rendre.
