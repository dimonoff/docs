# Créer un lieu

S’il s’agit de votre première connexion et que le projet n’a pas été configuré au préalable, il se peut qu’aucun lieu n’existe dans la plateforme. Si tel est le cas, le menu flottant sur la carte affichera un message vous l’indiquant, ainsi qu’une option pour créer votre premier lieu.

<figure><img src="../../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

En cliquant sur le bouton "Créer un nouveau lieu" (encadré en rouge dans l’image précédente), le menu flottant "Créer un nouveau lieu" apparaîtra afin de documenter son nom et son adresse pour l’afficher sur la carte. Une fois les informations entrées, veuillez cliquer sur le bouton "Soumettre".

<figure><img src="../../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>

Une fois soumis, le lieu apparaîtra dans le menu flottant sur la carte. Vous pourrez cliquer dessus sur la carte ou dans le menu afin de zoomer et d’en voir les informations en détails tel que ses zones ou ses données de stationnement en temps réel.

<figure><img src="../../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

Cette étape est décrite de façon plus détaillée dans la section "Créer des lieux et des zones" dans le chapitre "Inventaire".
