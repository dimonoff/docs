# Créer une zone

De la même façon que pour la création d’un premier lieu, lors de la création d’un lieu, l’option de créer sa première zone apparaît dans le menu flottant sur la carte. Un message l’indiquera avec une option pour créer votre première zone.

<figure><img src="../../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

En cliquant sur le bouton "Ajouter une zone à ce lieu" (encadré en rouge dans l’image précédente), le menu flottant "Nouvelle zone" apparaîtra afin de documenter son nom, le type de zone et son plan pour l’afficher sur la carte. \
Une fois les informations entrées, veuillez cliquer sur le bouton "Soumettre".

<figure><img src="../../.gitbook/assets/image (20).png" alt=""><figcaption></figcaption></figure>

Une fois soumise, la zone apparaîtra dans le menu flottant spécifique au lieu dans laquelle elle se trouve. Vous pourrez cliquer dessus dans le menu afin d’en voir les informations en détails tel que ses données de stationnement en temps réel.

Cette étape est décrite de façon plus détaillée dans la section "Créer des lieux et des zones" du chapitre "Inventaire".
