# Utilisateurs

La gestion des utilisateurs est faite à partir du menu 'Utilisateurs' dans le menu latéral gauche.

<figure><img src="../../.gitbook/assets/image (26).png" alt=""><figcaption></figcaption></figure>

La page principale de la section **Utilisateurs** permet de visualiser et de gérer l’ensemble des comptes ayant accès au projet. Elle constitue le point central pour superviser les permissions, l’activité et l’état des utilisateurs.

<figure><img src="../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

Cette page affiche un tableau regroupant les informations essentielles pour chaque utilisateur :

* **Email** : l’adresse courriel utilisée pour accéder à la plateforme.
* **Nom** : le nom complet de l’utilisateur.
* **Rôle** : le rôle attribué à l’utilisateur, définissant ses permissions dans le projet.
* **Invitation** : la date et l’heure auxquelles l’utilisateur a été invité sur le projet.
* **Statut** : l’état actuel de l’invitation ou du compte:

**En attente**: indique que l’utilisateur a bien reçu l’invitation, mais qu’il ne l’a pas encore acceptée.\
**Complété**: l’utilisateur a finalisé la configuration de son compte, notamment en choisissant un mot de passe, et qu’il a accepté le rôle qui lui a été assigné dans l’invitation.

**Filtrer les utilisateurs**

En haut du tableau, une **liste déroulante** intitulée **All Users** permet de filtrer l’affichage selon différents types d’utilisateurs (ex. : Project manager, Devices manager, Insight manager, etc.).\
![](<../../.gitbook/assets/image (56).png>)\
\
**Recherche d’un utilisateur**

Sur la droite, une **barre de recherche** est disponible pour retrouver rapidement un utilisateur.\
La recherche peut être effectuée à partir de l'adresse courriel ou le nom d'utilisateur\
\
![](<../../.gitbook/assets/image (57).png>)

\
**Créer un nouvel utilisateur**

1.  **Cliquer sur “New User”**

    <figure><img src="../../.gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>

    \\

    <figure><img src="../../.gitbook/assets/image (37).png" alt=""><figcaption></figcaption></figure>
2. **Remplir les champs du formulaire :**
   * **Rôle** : sélectionnez le rôle que vous souhaitez attribuer à l’utilisateur.
   * **Nom** : indiquez le nom de l’utilisateur.
   * **Adresse e-mail** : saisissez l’adresse e-mail à laquelle l’invitation sera envoyée.
   * **Message personnalisé (optionnel)** : vous pouvez ajouter un message pour accompagner l’invitation.
3. **Envoyer l’invitation**\
   Après avoir cliqué sur **“Créer un utilisateur”**, un e-mail d’invitation est automatiquement envoyé à l’adresse indiquée.\
   L’utilisateur recevra un **lien d’activation** lui permettant d’accepter l’invitation et de rejoindre le projet avec le rôle qui lui a été attribué.
