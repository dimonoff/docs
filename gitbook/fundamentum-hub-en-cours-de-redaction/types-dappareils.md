# Devices

**Devices** represent your physical connected objects (sensors, actuators, gateways, etc.) registered on the Fundamentum Hub platform.

#### Accessing Devices

To access the list of devices associated with the current registry:

1. Click on **Devices** in the main menu.
2. The list of devices for the selected registry is displayed.

<figure><img src="../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

#### Filter and Search Devices

The **Devices** page allows you to view and manage the devices in the registry. It includes the following features:

**Filters**\
Filter devices based on their type.

<figure><img src="../.gitbook/assets/image (14).png" alt=""><figcaption></figcaption></figure>

**Search**\
Use the search bar to quickly locate a specific device.

<figure><img src="../.gitbook/assets/image (13).png" alt=""><figcaption></figcaption></figure>

**Pagination**\
Pagination allows you to efficiently navigate through large lists of devices.

<figure><img src="../.gitbook/assets/image (15).png" alt=""><figcaption></figcaption></figure>



#### Create a New Device

To create a new device, follow these steps:

1. Click the **New Device** button.

<figure><img src="../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>

2. Select Device Type

<figure><img src="../.gitbook/assets/image (17).png" alt=""><figcaption></figcaption></figure>

3. Enter the required information:

* **Name**: Unique identifier for the device.
* **Serial Number**: Enter the serial number of the device.

<figure><img src="../.gitbook/assets/image (18).png" alt=""><figcaption></figcaption></figure>

4. Click **Validate** to confirm the creation of the device.

#### Configuration Tab

The **Configuration** tab allows you to view and manage the device’s settings.\
It displays the following elements:

* **General device information**
* **Status**
* **Version**
* **Configuration**

<figure><img src="../.gitbook/assets/image (21).png" alt=""><figcaption></figcaption></figure>

From the **Information** section, you can modify and edit the following fields:

* **Name**
* **Serial Number**
* **Device Type**
* **Protocols**
* **Communication Methods**

<figure><img src="../.gitbook/assets/image (22).png" alt=""><figcaption></figcaption></figure>

From the **Version** section, you can update the device firmware by clicking the **Upgrade** button.

<figure><img src="../.gitbook/assets/image (23).png" alt=""><figcaption></figcaption></figure>

From the **Configuration** section, you can send a configuration to a device in JSON format.

<figure><img src="../.gitbook/assets/image (24).png" alt=""><figcaption></figcaption></figure>



#### Commands Tab

The **Commands** tab allows you to manage the commands sent to the device.\
It offers the following features:

* **Command History**\
  Displays the list of commands sent to the device along with their status.
* **Send New Commands**\
  Allows you to send new commands to the device.
* **Command Indicator**\
  A badge shows the number of pending or recently sent commands.

<figure><img src="../.gitbook/assets/image (25).png" alt=""><figcaption></figcaption></figure>
