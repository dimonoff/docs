# Device types

**Device Types** define the categories and common characteristics shared by a group of similar devices. They help standardize device configuration and behavior.

#### Accessing Device Types

To access the list of device types associated with the current registry:

1. Click on **Device Types** in the main menu.
2. The list of device types for the selected registry is displayed.

<figure><img src="../.gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>



#### Create a New Device Type

To create a new device type, follow these steps:

1. Click the **New Device Type** button.

<figure><img src="../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

2. Enter the following information:

* **Name**: Descriptive name of the device type.
* **Icon**: Select a representative icon.
* **Asset Type ID**: Unique technical identifier for the device type.
* **Should Self Provision Has MPS :** Yes/NO
* **Is Accessible:** Yes/NO
* **Is a Gateway**: Check this option if the device type represents a gateway.
* **Is LoRaWAN:** Yes/NO
* **Default Device Configuration**

<figure><img src="../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

3. Click **Save** to confirm and register the new device type.



#### Edit a Device Type

To modify an existing device type, follow these steps:

1. Click the **Edit** button for the device type you want to modify.

<figure><img src="../.gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure>

2. Modify the desired settings.
3. Save the changes to apply the modifications.



**Note:** Generic device types can only be modified by administrators.<br>
